# VKyne Unity Tools Library (DLL)

A modular collection of reusable Unity utilities packaged as a compiled DLL for easy integration across projects.

---

## 🚀 Overview

This library contains a set of custom-built tools designed to streamline common development tasks in Unity projects. It is distributed as a DLL to:

* Protect internal implementation details
* Reduce project clutter
* Enable easy reuse across multiple projects
* Improve compile times in large Unity projects

---

## 🧰 Features

* 🔌 Plug-and-play DLL integration
* ⚡ Lightweight and dependency-conscious
* 🧱 Modular architecture (use only what you need)
* 🛠️ Built for real production workflows
* 🔄 Designed for reuse across multiple Unity projects

---

## 📦 Included Systems

* **Input Helpers**
  Simplified wrappers for Unity’s input handling

* **Utility Functions**
  Helper extensions and commonly used features

* **Data Structures**
  Optimized collections and helpers for game logic

* **Debug Tools**
  Runtime inspection helpers

---

## 📥 Installation

1. Copy the `.dll` file into your Unity project:

```
Assets/Plugins/
```

2. Unity will automatically recognize and compile the assembly

## 🧠 Design Philosophy

This library was built with a focus on:

* **Reusability** – Write once, use everywhere
* **Encapsulation** – Hide complexity behind clean APIs
* **Performance** – Avoid unnecessary allocations and overhead
* **Flexibility** – Systems are extensible via interfaces

---

## 🔒 Why a DLL?

Using a DLL instead of raw scripts provides:

* Cleaner project structure
* Faster iteration in large projects
* Protection of internal logic
* Easier versioning and distribution

---

## ⚠️ Notes

* Some systems may be simplified to remove project-specific dependencies
